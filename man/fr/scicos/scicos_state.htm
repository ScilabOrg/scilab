scicos_state - Définit une structure state

Module

  • Scicos

state

Une liste typée scilab de type xcs. Cette liste contient tous les états du diagramme qui vont évoluer pendant la simulation.
state contient x, z, oz, iz, tevts, evtspt, pointi et outtb.

Taille : 9.
Type : tlist scilab.

  • x
    Le registre des états continus qui est obtenu en concatenant tous les états continus de tous les blocs.

    Taille : total des tailles de tous les registres des états continus.
    Type : vecteur colonne de nombres réels.

  • z
    Le registre des états discrets qui est obtenu en concatenant tous les états discrets de tous les blocs.

    Taille : total des tailles de tous les registres des états discrets.
    Type : vecteur colonne de nombres réels.

  • oz
    La liste des états objets discrets obtenue par la concaténation de tous les états objets de tous les blocs.

    Taille : total des tailles de toutes les listes des états objet.
    Type : liste scilab.

  • iz
    Vecteur de taille égale au nombre de blocs.
    Ce vecteur est utilisé pour enregistrer les pointeurs des registres de travail (work) Si un bloc fait une allocation du registre work à l'initialisation (flag=4) alors le pointeur associé est enregistré ici.

    Taille : nombre de blocs.
    Type : vecteur colonne de nombres réels.

  • tevts
    Un vecteur de taille égale au nombre de sources d'activation. C'est une table contenant les dates des événements programmés dans evtspt.

    Taille : nombre de sources d'activation.
    Type : vecteur colonne de nombre entiers.

  • evtspt
    Un vecteur de taille égale au nombre de sources d'activation. C'est une table d'événements.

    Taille : nombre de sources d'activation.
    Type : vecteur colonne de nombre entiers.

  • pointi
    Le numéro du prochain événement.

    Taille : 1.
    Type : entier.

  • outtb
    Liste scilab contenant tous les registres de sortie des blocs. Chaque éléments de cette liste contient des données typées matricielles.

    Taille : nombre de ports réguliers de sortie.
    Type : liste scilab contenant des matrices.
