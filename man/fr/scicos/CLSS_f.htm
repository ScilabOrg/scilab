Description

Ce bloc réalise un système linéaire d'équations d'état en temporel continu.



Le système est défini par les matrices et par l'état initial . Les dimensions des matrices et de l'état initial doivent être appropriées.

Boîte de dialogue

  • matrix
    Une matrice carré.
    Propriétés : Type 'mat' de taille [-1,-1].

  • B matrix
    La matrice , [] si le système n'a pas d'entrées.
    Propriétés : Type 'mat' de taille ["size(%1,2)","-1"].

  • C matrix
    La matrice , [] si le système n'a pas de sorties.
    Propriétés : Type 'mat' de taille ["-1","size(%1,2)"].

  • D matrix
    La matrice , [] si le système n'a pas de terme D.
    Propriétés : Type 'mat' de taille [-1,-1].

  • Initial state
    Un état initial du système ( vectoriel ou scalaire).
    Propriétés : Type 'vec' de taille "size(%1,2)".

Propriétés par défaut

  • toujours actif: oui
  • direct-feedthrough: non
  • détection de passage à zéro: non
  • mode: non
  • entrée régulières:
    - port 1 : taille [1,1] / type 1
  • sorties régulières:
    - port 1 : taille [1,1] / type 1
  • nombre des entrées évènementielles: 0
  • nombre des sorties évènementielles: 0
  • possède un état continu: oui
  • possède un état discret: non
  • possède un état objet: non
  • nom de la fonction de calcul: csslti
