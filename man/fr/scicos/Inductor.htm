Description

Une bobine est un composant électrique qui peut stocker l'énergie dans des circuits électriques. Le rapport entre la tension aux borne d'une bobine d'inductance et le courant passant à travers le composant est donné par :

Des inductances peuvent également être utilisées pour distinguer les signaux haute fréquence des signaux de basse fréquence et ceci rend les inductances utiles dans la conception des filtres. Une inductance présente une forte impédance pour les signaux haute fréquence et est passante pour les signaux DC.
