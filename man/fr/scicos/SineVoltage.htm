Palette

  • Electrical.cosf - Boîte à outils des composants électriques

Description

Ce bloc est un bloc Modelica pour une source de tension sinusoïdale. La résistance ohmique interne de ce bloc est zéro.

Boîte de dialogue

  • Amplitude (Volt)
    Amplitude du signal sinosoïde.
    Propriétés : Type 'vec' de taille 1.

  • phase (rad)
    Phase du signal sinosoïde
    Propriétés : Type 'vec' de taille 1.

  • Frequency (Hz)
    Fréquence du signal sinosoïde
    Propriétés : Type 'vec' de taille 1.

  • Voltageoffset (V)
    Voltage offset du signal sinosoïde
    Propriétés : Type 'vec' de taille 1.

  • Timeoffset (s)
    Décalage de temps. Durant ce temps, le signal de sortie reste constant égal au voltage offset.
    Propriétés : Type 'vec' de taille 1.

Propriétés par défaut

  • Entrées :
    • Nom de la variable Modelica : 'p'
      Variable implicite.

  • Sorties :
    • Nom de la variable Modelica : 'n'
      Variable implicite.

  • Paramètres :
    • Nom du paramètre Modelica : 'V'
      Valeur par défaut : 1
      Variable d'état : non.

    • Nom du paramètre Modelica : 'phase'
      Valeur par défaut : 0
      Variable d'état : non.

    • Nom du paramètre Modelica : 'freqHz'
      Valeur par défaut : 1
      Variable d'état : non.

    • Nom du paramètre Modelica : 'offset'
      Valeur par défaut : 0
      Variable d'état : non.

    • Nom du paramètre Modelica : 'startTime'
      Valeur par défaut : 0
      Variable d'état : non.

  • Nom du fichier model : SineVoltage
