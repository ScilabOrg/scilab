Palette

  • Branching - Palette Branching

Description

Bloc spécial similaire au bloc If-Then-Else. Les entrées/sorties événementielles sont synchronisées. L'événement entrant est redirigé sur l'un des ports de sortie événementiels suivant le valeur de l'entrée régulière.

Boîte de dialogue

  • number of output event ports
    Un scalaire à valeur entière. Nombre de ports de sortie événementiel.
    Propriétés : Type 'vec' de taille 1.

  • Inherit (1: no, 0: yes)
    Si 'no' est sélectionné, alors le bloc est activé par son entrée événementiel. Si 'yes' est sélectionnée, alors le bloc est activé par les événements du signal reçu sur son port d'entrée régulier.
    Propriétés : Type 'vec' de taille 1.

  • zero-crossing (0: no, 1: yes)
    A sélectionner pour activer la détection de passage à zéro.
    Propriétés : Type 'vec' de taille 1.

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: oui
  • détection de passage à zéro: non
  • mode: non
  • entrée régulières:
    - port 1 : taille [1,1] / type -1
  • nombre des entrées évènementielles: 1
  • nombre des sorties évènementielles: 2
  • possède un état continu: non
  • possède un état discret: non
  • possède un état objet: non
  • nom de la fonction de calcul: eselect
