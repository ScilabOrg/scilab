Boîte de dialogue

  • State on halt
    : Type 'vec' de taille 1. Un scalaire à valeur réelle donnant l'état du bloc lors de l'arret. Cela permet de distinguer différents bloc 'STOP' entre eux (lors de la correction des 'bugs').

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: non
  • détection de passage à zéro: non
  • mode: non
  • nombre/taille des entrées régulières: 0 /
  • nombre/taille des sorties sorties régulières: 0 /
  • nombre/taille des entrées évènementielles: 1 / 1
  • nombre/taille des sorties évènementielles: 0 /
  • possède un état continu: non
  • possède un état discret: oui
  • nom de la fonction de calcul: hltblk

     &     rpar,nrpar,ipar,nipar,u,nu,y,ny)

c     Scicos block simulator
c     Notify simulation to stop  when called 
c     ipar(1) : stop reference
c
      double precision t,xd(*),x(*),z(*),tvec(*),rpar(*),u(*),y(*)
      integer flag,nevprt,nx,nz,ntvec,nrpar,ipar(*)
      integer nipar,nu,ny

c
      integer halt
      common /coshlt/ halt
c
      if(flag.eq.2) then
         halt=1
         if(nipar.gt.0) then
            z(1)=ipar(1)
         else
            z(1)=0.0d0
         endif
      endif
      end
