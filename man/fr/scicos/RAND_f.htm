Description

Ce bloc est un générateur de nombres aléatoires : chaque composante de sortie prend des valeurs constantes aléatoires par morceaux. Pour chaque événement les sorties prennent des nouvelles valeurs indépendantes.
La taille du port de sortie est déterminée par la taille des vecteurs A et B.

Boîte de dialogue

  • flag
    0 ou 1.
    0 pour une distribution uniforme sur [A,A+B].
    1 pour une distribution normale.
    Propriétés : Type 'vec' de taille 1.

  • A
    vecteur
    Propriétés : Type 'vec' de taille -1.

  • B
    vecteur
    Propriétés : Type 'vec' de taille -1.

  • seed
    scalaire
    Valeur de "graine" pour une séquence de nombre aléatoire.
    Propriétés : Type 'vec' de taille 1.

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: non
  • détection de passage à zéro: non
  • mode: non
  • sorties régulières:
    - port 1 : taille [1,1] / type 1
  • nombre des entrées évènementielles: 1
  • nombre des sorties évènementielles: 0
  • possède un état continu: non
  • possède un état discret: oui
  • possède un état objet: non
  • nom de la fonction de calcul: rndblk
