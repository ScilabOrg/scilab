scicos_link - Définit une structure link

Module

  • Scicos

link

Size : 8.
Type : liste scilab.

  • xx
    Un vecteur de coordonnées des abscisses x d'un lien. Un lien est définit comme une polyline.

    Taille : nombre total de points du liens.
    Type : vecteur colonne de nombre réels.

  • yy
    Un vecteur de coordonnées des ordonnées y d'un lien. Un lien est définit comme une polyline.

    Taille : nombre total de points du liens.
    Type : vecteur colonne de nombre réels.

  • id
    Une chaîne de caractères définissant l'identification du lien.

    Taille : 1.
    Type : chaîne de caractères.

  • thick
    Un vecteur de taille deux définissant l'épaisseur du lien.

    Taille : 2.
    Type : vecteur ligne de nombre entiers.

  • ct
    La première entrée de ce vecteur désigne la couleur et la deuxième la nature du lien. La valeur de cette dernière entrée est 1 pour un lien régulier, -1 pour un lien d'activation et 2 pour un lien implicite.

    Taille : 2.
    Type : vecteur ligne de nombre entiers.

  • from
    Un vecteur de taille 3 qui designe un numéro de bloc, un numéro de port et un type de port (0 pour une sortie et 1 pour une entrée) concernant l'origine du lien. Notez que la troisième entrée peut être 1 si le lien est implicite. Sinon il est zéro.

    Taille : 3.
    Type : vecteur ligne de nombre entiers.

  • to
    Un vecteur de taille 3 qui designe un numéro de bloc, un numéro de port et un type de port (0 pour une sortie et 1 pour une entrée) concernant la destination du lien. Notez que la troisième entrée peut être 1 si le lien est implicite. Sinon il est zéro.

    Taille : 3.
    Type : vecteur ligne de nombre entiers.
