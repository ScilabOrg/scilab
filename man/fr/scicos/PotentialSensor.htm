Palette

  • Electrical.cosf - Boîte à outils des composants électriques

Description

Ce bloc est employé pour mesurer la tension d'un noeud par rapport à celle de terre (zero) dans un circuit électrique. La tension est donnée à la partie explicite du modèle par un port explicite.

Propriétés par défaut

  • Entrées :
    • Nom de la variable Modelica : 'p'
      Variable implicite.

  • Sorties :
    • Nom de la variable Modelica : 'v'
      Variable explicite.

  • Nom du fichier model : PotentialSensor
