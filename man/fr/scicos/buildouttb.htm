buildouttb - Construction d'une liste vide %cpr.state.outtb

Module

  • Scicos

Description

Construit une liste "outtb" initialisée.

Séquence d'appel 

[outtb]=buildouttb(lnksz,lnktyp)

Paramètres

  • outtb : une liste de taille n.
  • lnksz : Ce paramètre donne la taille d'un objet scilab dans la liste outtb.
    Cette matrice de nombres entiers ou de nombres réels peut avoir une taille n,2 ou 2,n.
  • lnktyp : Ce paramètre donne le type d'objet scilab dans outtb :
    1 : réel
    2 : complexe
    3 : int32
    4 : int16
    5 : int8
    6 : uint32
    7 : uint16
    8 : uint8
    sinon : réel
    Cette matrice de nombres entiers ou de nombres réels peut avoir une taille n,1 ou 1,n.
