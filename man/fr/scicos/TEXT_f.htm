Palette

  • Others - Palette Others

Description

Ce bloc spécial est utilisée pour ajouter du texte dans un diagramme scicos. Il n'affecte pas la simulation.

Boîte de dialogue

  • Text
    la chaîne de caractères à afficher.
    Propriétés : Type 'str' de taille -1

  • Font number
    un entier positif (plus petit que 6) donnant le type de la police.
    Propriétés : Type 'vec' de taille 1

  • Font size
    un entier positif donnant la taille de la police (voir xset).
    Propriétés : Type 'vec' de taille 1

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: non
  • détection de passage à zéro: non
  • mode: non
  • nombre/taille des entrées régulières: 0 /
  • nombre/taille des sorties sorties régulières: 0 /
  • nombre/taille des entrées évènementielles: 0 /
  • nombre/taille des sorties évènementielles: 0 /
  • possède un état continu: non
  • possède un état discret: non
  • nom de la fonction de calcul: text

Fonction d'interface

  • SCI/macros/scicos_blocks/Misc/TEXT_f.sci [voir code]
