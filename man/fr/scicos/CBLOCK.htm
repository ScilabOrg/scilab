Boîte de dialogue

  • simulation function
    : Type 'str' de taille 1. Nom de la fonction qui doit être générée.
  • is block implicit? (y,n)
    : Type 'str' de taille 1. Si 'yes' est utilisée, alors le solver implicite (dasrt) est utilisé, sinon c'est le solveur explicite (lsodar) qui est utilisé.
  • input ports sizes
    : Type 'vec' de taille -1. Nombre de ports réguliers en entrée.
  • output ports sizes
    : Type 'vec' de taille -1. Nombre de ports réguliers de sortie.
  • input event ports sizes
    : Type 'vec' de taille -1. Nombre de ports d'activation en entrée.
  • output events ports sizes
    : Type 'vec' de taille -1. Nombre de ports d'activation de sortie.
  • initial continuous state
    : Type 'vec' de taille -1. Conditions initiales du vecteur d'état continu.
  • number of zero crossing surfaces
    : Type 'vec' de taille 1. A sélectionner pour utiliser les détections de traversée de zéro.
  • initial discrete state
    : Type 'vec' de taille -1. Conditions initiales du vecteur d'état discret.
  • Real parameters vector
    : Type 'vec' de taille -1. Vecteur des paramètres réels.
  • Integer parameters vector
    : Type 'vec' de taille -1. Vecteur des paramètres entiers.
  • initial firing vector ( 0 for no firing)
    : Type 'vec' de taille "sum(%6)". Un vecteur dont la taille correspond au nombre de ports d'activation de sortie. Donne la premère dae d'activation de ceux ci.
  • direct feedthrough (y or n)
    : Type 'str' de taille 1. L'entrée du bloc au temps courant détermine la sortie du bloc au temps courant. Cela force l'entrée à être en sortie, comme si le système était en régime établi.
  • time dependence (y or n)
    : Type 'str' de taille 1. Crée un signal qui est spécifié par la dépendance au temps. (pas continu)

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: oui
  • détection de passage à zéro: non
  • mode: non
  • nombre/taille des entrées régulières: 1 / 1
  • nombre/taille des sorties sorties régulières: 1 / 1
  • nombre/taille des entrées évènementielles: 0 /
  • nombre/taille des sorties évènementielles: 0 /
  • possède un état continu: non
  • possède un état discret: non
