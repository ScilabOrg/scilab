Boîte à outils des composants thermohydrauliques

Module

  • Scicos

Description

La boîte à outils de thermohydraulique contient certains composants thermohydrauliques tels que la source de pression, le tuyau, les vannes réglantes, etc.

Blocs

  • Bache - Bache thermohydraulique
  • PerteDP - Perte de charge thermohydraulique (Tuyau)
  • PuitsP - Puits thermohydraulique de pression constante
  • SourceP - Source thermohydraulique de pression constante
  • VanneReglante - Vanne réglante thermohydraulique
