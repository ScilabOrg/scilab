Boîte à outils des composants électriques

Module

  • Scicos

Description

La boîte à outils électrique contient les composants basiques électriques tels que la source de tension, la diode, le condensateur, etc.

Blocs

  • Capacitor - Condensateur électrique
  • ConstantVoltage - Source de tension électrique constante
  • CurrentSensor - Capteur de courant électrique
  • Diode - Diode électrique
  • Ground - Terre (potentiel zero)
  • Inductor - Inductance électrique
  • NPN - NPN transitor
  • PotentialSensor - Capteur de potentiel électrique
  • Resistor - Résistance électrique
  • VariableResistor - Résistance électrique variable
  • VoltageSensor - Capteur de tension électrique relative
  • VsourceAC - Source de tension électrique AC
  • VVsourceAC - Source contrôlable de tension électrique AC
