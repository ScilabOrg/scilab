Palette

  • OldBlocks - Palette des anciens blocs

Description

Le bloc somme réalise l'addition de ses entrées. Ce bloc peut additionner des scalaires ou des vecteurs d'entrée.

Boîte de dialogue

  • Inputs ports signs/gain
    Définit le signe de l'addition ainsi qu'un gain pour chaque entrée.
    Propriétés : Type 'vec' de taille -1.

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: oui
  • détection de passage à zéro: non
  • mode: non
  • entrée régulières:
    - port 1 : taille [-1,1] / type 1
    - port 2 : taille [-1,1] / type 1
  • sorties régulières:
    - port 1 : taille [-1,1] / type 1
  • nombre des entrées évènementielles: 0
  • nombre des sorties évènementielles: 0
  • possède un état continu: non
  • possède un état discret: non
  • possède un état objet: non
  • nom de la fonction de calcul: sum
