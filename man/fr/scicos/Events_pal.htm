Palette Events

Module

  • Scicos

Description

La palette 'Events' (événements) est utilisée pour manipuler les événements sur le diagramme. Elle contient entre autre plusieurs sources d'activation, des blocs synchrones ainsi que des blocs pour regrouper/combiner les événements sur un seul lien.

Blocs

  • ANDBLK - Et activation
  • ANDLOG_f - ET logique
  • CLKSOMV_f - Union d'activations
  • CLOCK_c - Horloge d'activation
  • ESELECT_f - Bloc synchrone Event-Select
  • EVTDLY_f - Retard événementiel (Ancien bloc)
  • EVTGEN_f - Générateur d'événement
  • EVTVARDLY - Retard variable événementiel
  • freq_div - Diviseur de fréquence
  • HALT_f - Halt
  • IFTHEL_f - Bloc synchrone If-Then-Else
  • M_freq - Horloge à fréquence multiples
  • SampleCLK - Horloge synchrone
