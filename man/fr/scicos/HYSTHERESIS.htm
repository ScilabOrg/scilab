Boîte de dialogue

  • switch on at
    : Type 'vec' de taille 1. Un scalaire à valeur réelle donnant la valeur du point de commutation ouvert. La donnée en entrée est convertie en utilisant une approximation de type 'arrondie au plus proche'.
  • switch off at
    : Type 'vec' de taille 1. Un scalaire à valeur réelle donnant la valeur du point de commutation fermé. La donnée en entrée est convertie en utilisant une approximation de type 'arrondie au plus proche'.
  • output when on
    : Type 'vec' de taille 1. Un scalaire à valeur réelle donnant la valeur en sortie du bloc lorsque le cycle est à l'état ouvert.
  • output when off
    : Type 'vec' de taille 1. Un scalaire à valeur réelle donnant la valeur en sortie du bloc lorsque le cycle est à l'état fermé.
  • use zero crossing: yes (1), no (0)
    : Type 'vec' de taille 1. A sélectionner pour permettre la détection de zéro.

  • toujours actif: non

HYSTHERESIS.sci

Fonction de calcul (type 4) 


#include "scicos_block.h"
#include <math.h>

void  hystheresis(scicos_block *block,int flag)
{
  if (flag==1){
    if (get_phase_simulation()==1||block->ng==0) {
      if (*block->inptr[0]>=block->rpar[0]){
	block->outptr[0][0]=block->rpar[2];
      }else if (*block->inptr[0]<=block->rpar[1]){
	block->outptr[0][0]=block->rpar[3];
      }
    }else{
      if (block->mode[0]<2){
	block->outptr[0][0]=block->rpar[3];
      }else{
	block->outptr[0][0]=block->rpar[2];
      }
    } 
  } else if (flag==9){
    block->g[0]=*block->inptr[0]-(block->rpar[0]);
    block->g[1]=*block->inptr[0]-(block->rpar[1]);
    if (get_phase_simulation()==1) {
      if (block->g[0]>=0){
	block->mode[0]=2;
      }else if (block->g[1]<=0){
	block->mode[0]=1;
      }
    }
  }
}
