Palette

  • Sinks - Palette Affichage

Description

Ce bloc est utilisé pour transférer des données simulées dans l'environnement Scilab.
A chaque instants d'échantillonnage, aussi bien les dates que les valeurs de l'entrée sont enregistrées.

Boîte de dialogue

  • Size of buffer
    Définit la taille du buffer. Cela donne le nombre total d'échantillons qui seront enregistrés pendant la simulation.
    Ce buffer est circulaire.
    Propriétés : Type 'vec' de taille 1.

  • Scilab variable name
    Définit le nom de la variable Scilab. Ce nom doit être un nom de variable valide.
    La simulation doit être achevée pour pouvoir récupérer la variable dans Scilab.
    Propriétés : Type 'str' de taille 1.

  • Inherit (no:0, yes:1)
    Option pour choisir l'activation par héritage d'événement via le port d'entrée régulier ou explicitement par le port d'entrée événementiel.

    Propriétés : Type 'vec' de taille 1.

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: non
  • détection de passage à zéro: non
  • mode: non
  • entrée régulières:
    - port 1 : taille [-1,1] / type -1
  • nombre des entrées évènementielles: 1
  • nombre des sorties évènementielles: 0
  • possède un état continu: non
  • possède un état discret: non
  • possède un état objet: non
  • nom de la fonction de calcul: towork_c
