Description

This block realizes a shift register. At every input event, the register is shifted one step. This block support the integers data types.

Dialog box

  • Register initial condition
    : Type 'vec' of size -1. It contains the initial state of the register.
  • Datatype(3=int32 4=int16 5=int8 ...)
    : Type 'vec' of size 1. It indicates the type of the input/output data. It support all the integer datatype, number must be between 3 and 8.

Default properties

  • always active: no
  • direct-feedthrough: no
  • zero-crossing: no
  • mode: no
  • number/sizes of inputs: 1 / 1
  • number/sizes of outputs: 1 / 1
  • number/sizes of activation inputs: 1 / 1
  • number/sizes of activation outputs: 0 /
  • continuous-time state: no
  • discrete-time state: no
  • name of computational function: delay4_i32

Computational function (type 4) 


#include "scicos_block4.h"

void delay4_i32(scicos_block *block,int flag)
{
  /* Copyright INRIA
   
     Scicos block simulator
     Ouputs nx*dt delayed input */

  long *y,*u,*oz;
  int nz,mz;
  int i;
  
  u=Getint32InPortPtrs(block,1);
  y=Getint32OutPortPtrs(block,1);
  oz=Getint32OzPtrs(block,1);
  nz=GetOzSize(block,1,2);
  mz=GetOzSize(block,1,1);

  if ((flag==1)||(flag==6)||(flag==4)){
    y[0]=oz[0];
  }else if (flag == 2){
    /*  shift buffer */
    for (i=0; i<=(mz*nz)-2; i++){
      oz[i]=oz[i+1];
    }
    /* add new point to the buffer */
    oz[(mz*nz)-1]=u[0];
  }
}
