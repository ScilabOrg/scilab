Description

This blocks computes the conjugate of a complex input matrix.

Example 

--                  --               --                  --
| 1+i   2+3i    3+2i |  Conjugate    | 1-i   2-3i    3-2i |
| 4+i   5-8i    6-2i | ----------->  | 4-i   5+8i    6+2i |
--                  --               --                  --
