Description

This block realizes a shift register. At every input event, the register is shifted one step.

Dialog box

  • Register initial condition: A column vector. It contains the initial state of the register.

Default properties

  • always active: no
  • direct-feedthrough: no
  • zero-crossing: no
  • mode: no
  • number/sizes of inputs: 1 / 1
  • number/sizes of outputs: 1 / 1
  • number/sizes of activation inputs: 1 / 1
  • number/sizes of activation outputs: 0 /
  • continuous-time state: no
  • discrete-time state: yes
  • name of computational function: delay

Interfacing function
