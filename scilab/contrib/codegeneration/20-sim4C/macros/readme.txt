How to manually compile the 20-sim 4C macros:
---------------------------------------------------------

1) Go to the 20-sim 4C macro directory by the following command within the Scilab console:
	-->cd "C:\Program Files\scilab-5.2.2\contrib\20-sim4C\macros"
	Change the installation directory according to your configuration
	
2) Build the macros:
	-->exec("buildmacros.sce")

3) Start Xcos:
	-->xcos
	
4) Try the 20-sim 4C codegeneration by "tools\20-sim 4C codegeneration"
