What's new in Scilab 5.4.0?

This version introduces Scilab desktop which allows the user to save and restore his Scilab windows configuration. In this context, strong efforts have been carried out on the graphical components:

  • File browser to facilitate quick access to Scilab files and documents.
  • Browsevar look and feel has been improved and new features introduced.
  • Editvar renewed which now allows to plot and manage almost all Scilab variable types and is much more faster.
  • History browser now faster and more efficient.

This release also introduces Scilab preferences management through a unique interface to configure Scilab and its modules.

An internal improvement of Scilab graphics, aiming at fixing graphics card portability issues, provides faster and better 2-D and 3-D plotting. Graphic export features have been improved (portability, offscreen rendering, etc).

A new Scilab file format called Scilab Open Data format (SOD) is now used by save and load functions. SOD format is a documented and fully tested HDF5-based standard.

XML access from Scilab has never been easier thanks to the new dedicated module to read and write XML files.

And also, new "image" and "table" (read-only) uicontrol styles, new eigs function to calculate eigenvalues and eigenvectors of matrices, including sparse matrices, refactored CSV functions to improve writing/reading performances from a 3x to 200x factor.

Improvements in Scilab documentation: The help pages on graphics now contain a sample image, some chapters are better categorized, etc.

Under Windows, Scilab can now be installed without administration rights.
