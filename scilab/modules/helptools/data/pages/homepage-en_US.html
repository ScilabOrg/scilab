What's new in Scilab 5.4.0?

This new version introduces Scilab Desktop. Scilab desktop is the ability preserve and restore the layouts of Scilab.
In this context, strong efforts have been carried out on the graphical compoments:

  • A file browser component is now offered to Scilab users. Quick access to Scilab files and documents are now easier.
  • Browsevar look and feel has been improved and new features introduced.
  • Editvar had a important refreshment. It allows plot and subplot, manages almost all Scilab variables and is as fast as the light.
  • History browser is now faster and easier.

XML: a new module to read and write XML files is proposed to users. Thanks to the XPath syntax and its Scilab list mapping, XML access from Scilab has never been easier.

Available in a future alpha or beta version of Scilab 5.4.0, a big internal improvement of the Scilab graphic will provided faster and better 2D and 3D plotting.
A future development release will also introduce Scilab preferences (A unified way to configure Scilab and modules).
